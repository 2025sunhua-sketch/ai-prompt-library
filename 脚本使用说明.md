# 📝 脚本使用说明

## 🎯 问题诊断

你遇到的情况：运行 `init-git.bat` 时系统直接关闭，没有跳出输入界面。

**根本原因**：
1. 脚本在某些错误分支缺少 `pause` 命令
2. 或者脚本执行完毕后自动关闭窗口
3. 或者遇到了权限问题

### 方案 C：手动执行命令（最可靠）

如果两个脚本都无法运行，可以手动执行：

**步骤**：

1. **打开命令行**
   - 在项目文件夹空白处
   - 按住 `Shift` + 右键
   - 选择"在此处打开 PowerShell 窗口"

2. **逐行执行以下命令**：

```powershell
# 1. 检查 Git
git --version

# 2. 初始化仓库
git init

# 3. 配置你的信息（替换为你的）
git config user.name "你的 GitHub 用户名"
git config user.email "你的邮箱@example.com"

# 4. 添加文件
git add .

# 5. 提交
git commit -m "Initial commit: AI Prompt Library website"

# 6. 重命名分支
git branch -M main

# 7. 添加远程仓库（二选一）
# HTTPS 方式：
git remote add origin https://github.com/你的用户名/ai-prompt-library.git

# 或 SSH 方式（推荐）：
git remote add origin git@github.com:你的用户名/ai-prompt-library.git

# 8. 推送
git push -u origin main
```
