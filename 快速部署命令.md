# GitHub Pages 快速部署命令清单

> **打印此页贴在桌边，随时参考！**

---

## 🚀 首次部署（按顺序执行）

```bash
# Step 1: 进入项目目录（在文件资源管理器右键 → Git Bash Here）

# Step 2: 初始化仓库
git init

# Step 3: 配置用户信息（替换为你的真实信息）
git config user.name "孙启涵"
git config user.email "你的邮箱@example.com"

# Step 4: 添加所有文件
git add .

# Step 5: 提交首次 commit
git commit -m "Initial commit: AI 提示词库网站 v1.0"

# Step 6: 创建远程仓库后，关联并推送
git remote add origin https://github.com/你的用户名/ai-prompt-library.git
git branch -M main
git push -u origin main
```

---

## 🔄 日常更新（每次修改代码后）

```bash
# Step 1: 查看变更
git status

# Step 2: 添加变更
git add .

# Step 3: 提交
git commit -m "Update: 描述你的修改"

# Step 4: 推送
git push
```

## 🔑 身份验证方式

### 方式 A：Personal Access Token（推荐新手）

1. 访问：https://github.com/settings/tokens
2. 生成 token（勾选 `repo` 权限）
3. 复制 token（只显示一次！）
4. Git 推送时：
   - Username: 你的 GitHub 用户名
   - Password: **粘贴 token**（不会显示字符，正常现象）

### 方式 B：SSH 密钥（推荐长期）

```bash
# 生成密钥
ssh-keygen -t ed25519 -C "你的邮箱@example.com"

# 查看公钥
cat ~/.ssh/id_ed25519.pub

# 复制输出内容到 GitHub: https://github.com/settings/keys
```

## 🌐 启用 GitHub Pages

1. 访问仓库 Settings → Pages
2. Source: Deploy from a branch
3. Branch: main / root
4. Save → 等待 2-5 分钟

**网站地址格式：**
```
https://你的用户名.github.io/ai-prompt-library/
```

## 🛠️ 常用命令速查

```bash
# 查看当前状态
git status

# 查看提交历史
git log --oneline

# 撤销上一次的 commit（保留修改）
git reset HEAD~1

# 强制推送到远程（谨慎使用）
git push -f origin main

# 查看远程仓库地址
git remote -v

# 更换远程仓库地址
git remote set-url origin https://github.com/新地址.git
```

## ⚠️ 常见问题一行解决

```bash
# 问题：推送被拒绝
git pull origin main --rebase
git push

# 问题：想忽略已跟踪的文件
git rm -r --cached node_modules

# 问题：修改上一次提交信息
git commit --amend -m "新的提交信息"

# 问题：查看分支
git branch -a
```
